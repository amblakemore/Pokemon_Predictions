Model Deployment

Presented Machine Learning Algorithm was created using Google Colaboratory (Colab). Colab notebooks execute arbitrary python code on Google's cloud servers and and is especially well suited to machine learning processes. Therefore, give us opportunity to traing our model with more than 7,000 images.

To use trained model in our website, the model was saved in .h5 format. This saved model was Then it was loaded with Tensorflow, Keras and Flask. Note: versions of Tensorflow and Keras in the virtual environment (where flask python file will be excecuted) should be the same versions (or higher) as in Colab where model was trained and saved.

To make sure that our model was saved correctly and it is the same model from the Colab notebook, the weights of original model and saved copy were compared.

After this, model was loaded and used for predictions. Check Demo page to try it!!!
